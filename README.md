Introduction:
This project is a Simple Shopping/Billing System developed using C++ programming language. It allows users to select products from a menu, enter the quantity, and calculate the total bill. The program uses basic concepts like loops, switch-case, and user input/output to simulate a real-world shopping experience.

Objective of the Project:
To design a simple shopping/billing system using C++ that calculates the total cost of selected products by applying OOP concepts like encapsulation, inheritance, and polymorphism.

System Design:
1. Input
User selects a product (choice from 1 to 5).
User enters the quantity of the selected product.

2. Process
Display product menu
Take user choice
Create object of selected product
Accept quantity
Calculate total using calculateTotal()
Add to overall total
Repeat until exit

3. Output
Displays the total bill amount after the user selects "Exit".

4. Tools & Technologies
Programming Language: C++
Concepts Used:
Encapsulation
Inheritance
Polymorphism
Classes & Objects

Conclusion:
This project demonstrates how OOP concepts can be used to build a modular and reusable billing system. It improves code structure, reduces repetition, and makes the program easy to extend with new products.
