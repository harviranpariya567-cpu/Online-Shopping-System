Introduction:
This project is a Simple Shopping/Billing System developed using C++ programming language. It allows users to select products from a menu, enter the quantity, and calculate the total bill. The program uses basic concepts like loops, switch-case, and user input/output to simulate a real-world shopping experience.

Objective of the Project:
To understand the use of C++ control structures like do-while loop and switch-case.
To create a simple menu-driven program.
To perform basic arithmetic operations for billing.
To improve skills in user interaction and program flow control.
To simulate a real-life shopping system in a basic way.

System Design:
1. Input
User selects a product (choice from 1 to 5).
User enters the quantity of the selected product.

2. Process
Program displays a menu repeatedly using a do-while loop.
Based on the user’s choice, the program:
Asks for quantity
Calculates cost using price × quantity
Adds it to the total amount
Uses switch-case to handle different product options.

3. Output
Displays the total bill amount after the user selects "Exit".

4. Tools & Technologies
Programming Language: C++
Concepts Used:
Loop (do-while)
Conditional Statements (switch-case)
Variables and Data Types
Input/Output (cin, cout)

Conclusion:
This project demonstrates how a simple billing system can be created using basic C++ concepts. It helps in understanding program flow, user input handling, and calculations. Although it is a basic system, it can be further improved by adding features like item names in the bill, discounts, taxes, and file handling for storing records.
